# Unreal-Shooting-Game

고정된 위치에 있는 보스의 패턴을 주어진 스킬을 활용하여 격파하는 탄막 슈팅 게임

## How to play
이동 : WASD
공격력 향상 : A
무적 : S

보스 근처에 가면 자동으로 공격탄이 보스에게 적중.

주어진 스킬을 활용해 보스 근처에서 버티는 것이 공략.
